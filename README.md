Nombre del Proyecto: Cinelandia

Descripción: Este proyecto es una página web básica desarrollada como parte de un curso de Front-End. La página está estructurada con HTML semántico y utiliza las etiquetas `<header>`,`<main>`, y `<footer>` para organizar el contenido. El objetivo es aprender a crear la estructura básica de una página web y prepararla para futuras mejoras con CSS y JavaScript.

Qué hace el proyecto: El sitio es una web para ver reviews y noticias de series y películas.

Por qué el proyecto es útil: Porque comunica a los cinéfilos o gente interesada en ver películas/series las reseñas de las mismas ayudando a elegir cual es la mejor opcion para ver en base a sus gustos y los comentarios de los críticos creadores de la web, así como también brindando las últimas noticias sobre series y películas.

Quién mantiene y contribuye con el proyecto: Miguel Pavon, estudiante del curso "Front end JS" lleva adelante este proyecto.
